# Banking Data Analysis Power BI Dashboard

This project presents an interactive Power BI dashboard designed to analyze banking data and uncover insights related to customer demographics, loan distribution, and financial performance.

The dashboard provides a clear view of key banking metrics and helps identify patterns in customer behavior and financial transactions.

## Tools & Technologies

* Microsoft Power BI
* Data Cleaning
* Data Visualization
* CSV Dataset

## Key Insights

* Total number of customers
* Loan distribution across different customer segments
* Customer demographics analysis
* Balance and transaction trends
* Region-wise banking activity

## Dataset

The dataset used in this project contains banking information such as customer details, account balances, loans, and financial transactions.

## Project Objective

The objective of this project is to analyze banking data and convert raw financial information into meaningful business insights using interactive Power BI visualizations.
